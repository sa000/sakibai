# SAKIB AI

Modern consulting website for AI, machine learning, and data science services.

**Live Site**: https://sakibai.com

## Tech Stack

- HTML5, CSS3, JavaScript
- Web3Forms for contact functionality
- Deployed via GitHub Actions to GitHub Pages

## Local Development

```bash
python -m http.server 8000
```

Visit http://localhost:8000

## Deployment

Site automatically deploys to GitHub Pages when changes are pushed to `main` branch.

See `DEPLOYMENT.md` for details.
